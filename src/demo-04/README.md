👍 那我帮你写一个 条件变量（std::condition_variable）的生产者-消费者示例。这个例子能直观展示线程之间的 同步协作：生产者生产数据，消费者等待数据，有数据就消费。

🧪 实验 4：生产者-消费者（条件变量版）

🔑 要点

1. cv.wait(lock, predicate)

- 线程会等待，直到条件满足（队列非空或生产结束）。

- 避免忙等，提高效率。

2. cv.notify_one() / cv.notify_all()

- 通知一个/所有等待的线程去检查条件。

3. 互斥锁 + 条件变量配合

- 锁保护共享资源（队列），

- 条件变量负责线程间的“唤醒”。

📌 到这里，你就掌握了：

- 进程 vs 线程的差异（独立 vs 共享内存）

- 互斥锁（避免数据竞争）

- 条件变量（线程间协作）
